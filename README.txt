Задача 1:
Напишите на С++ http-сервис, который позволит вести на сервере библиотеку книг,
интересуют только название книги и автор.
Сервис должен предоставлять REST API для получения/создания/обновления/удаления
записей в библиотеке, все указанные операции должные производиться через хранимые
функции/процедуры в PostgreSQL.
Кроме того API должно:
• не позволять создавать книги без авторов
• не позволять создавать авторов без книг
• возвращать все книги конкретного автора;
• возвращать всех авторов конкретной книги;
• возвращать список всех книг у которых более заданого количества авторов.
